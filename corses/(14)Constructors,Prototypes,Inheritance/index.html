    Custom JavaScript objects

    1. The class

    JavaScript has no classes
    statement

    JavaScript uses functions as classes

    
			var Person = function () {};

    Object creation

    			var person1 = new Person();
			var person2 = new Person();

    Constructor

    • Sets the object's properties
    • Calls methods to prepare the object for use

    Constructor Example

    			var Person = function () {
				console.log('instance created');
			};
			var person1 = new Person();
			var person2 = new Person();

    Constructor

    • There is no object being created explicitly.
    • The properties and method are assigned directly onto the this object.
    • There is no return statement.

    Properties

    			var Person = function (firstName) {
				this.firstName = firstName; // add properties
				console.log('Person instantiated');
			};
			var person1 = new Person('Alice');
			var person2 = new Person('Bob');
			console.log('person1 is ' + person1.firstName); // logs "person1 is Alice"
			console.log('person2 is ' + person2.firstName); // logs "person2 is Bob"

    this

    Refers to the current object, lets you work with properties from within the object

    new

    1. Create a new object.
    2. Assign the this value of the constructor to the new object (so this points to the new object).
    3. Execute the code inside the constructor (adds properties to the new object).
    4. Return the new object.

    Methods

    • Functions of object, it's behavior
    • To define a method, assign a function to a named property

    Methods Example

    			var Person = function (firstName) {
				this.firstName = firstName;
				this.sayName = function(){  // add methods
					console.log(this.firstName);
				};
			};
			var person1 = new Person('Alice');
			person1.sayName(); // logs "Alice"

    Methods Example Problem

    			var person1 = new Person('Alice');
			var person2 = new Person('Bob');
			person1.sayName == person2.sayName;  // false

    Memory consuming

    Prototype

    • Each function is created with a prototype property
    • Object containing properties and methods
    • Available to instances of a particular reference type

    Prototype Example

    			
			Person.prototype.sayName = function(){  // add to prototype
					console.log(this.firstName);
			};
			Person.prototype.planet = 'Earth';
			var person1 = new Person('Alice');
			person1.sayName(); // logs "Alice"
			console.log(person1.planet); // logs "Earth"

    2. Inheritance

    Inheritance

    is a way to create a class as a specialized version of one or more classes

    Inheritance

    In JavaScript you do this by assigning an instance
    of the parent class to the child class, and then specializing

    Object.create()

    Creates a new object with the specified prototype object and properties.

    			Object.create(proto[, propertiesObject])
			var parent = {
				name: "Bob",
				age: 45
			};
			var child = Object.create(parent);

    Inheritance: Constructor Stealing

    			function Student(firstName, subject) {
				// Call the parent constructor, making sure (using Function#call)
				// that "this" is set correctly during the call
				Person.call(this, firstName);
				// Initialize our Student-specific properties
				this.subject = subject;
			};

    Inheritance: Prototypal

    			Person.prototype.sayName = function(){  // add to prototype
					console.log(this.firstName);
			};
			Person.prototype.walk = function(){  // add to prototype
					console.log('I'm walking!');
			};
			Person.prototype.planet = 'Earth';

    Inheritance: Prototypal

    			Student.prototype = Object.create(Person.prototype);
			// Set the "constructor" property to refer to Student
			Student.prototype.constructor = Student;
			Student.prototype.sayHello = function(){
				console.log("Hello, I'm " + this.firstName + ". I'm studying " + this.subject);
			};
			Student.prototype.sayGoodBye = function(){
				console.log("Goodbye!");
			};

    Inheritance: Prototypal

    			var student1 = new Student("Janet", "Applied Physics");
			student1.sayHello();   // "Hello, I'm Janet. I'm studying Applied Physics"
			student1.walk();       // "I am walking!"
			student1.sayGoodBye(); // "Goodbye!"
			// Check that instanceof works correctly
			console.log(student1 instanceof Person);  // true 
			console.log(student1 instanceof Student); // true
