Model

Відповідає за збереження предметно-
орієнтованих даних та бізнес-логіки
програми.

Model

Сповіщує своїх "спостерігачів" про
стан даних.

Model

Нічого не знає про представлення
та контроллери.

Змінюється за допомогою контроллера.

View

Відповідає за представлення даних з моделі
та визначає як відобразити дані користувачу.

View

Взаємодіє з користувачем та передає
інформацію про дії користувача до контроллера.

View

Підписується на модель і оновлює представлення
при змінах в моделі.

Controller

Відповідає за зв'язок користувача із програмою.

Виступає посередником між
моделями та представленнями.

Controller

Додає обробники подій до представлення
та оновлює модель коли юзер маніпулює видом.

Controller

Залежить від моделі.

Rivets.js

Шо за Rivets.js?

Lightweight and powerful data binding + templating solution
for building modern web applications.

Declarative data binding is what all the cool kids are doing
right now.

Як би ви реалізували таку поведінку без MVC?

Створюємо власні байндери

One-way binders

				rivets.binders.color = function(el, value){
					el.style.color = value;
				}
<button rv-color="label.color">Apply</button>

Two-way binders

				rivets.binders.['my-binder'] = {
				  bind: function(el){
                      model = this.model
                      el.addEventListener('input', this.publish)
                  }
                  routine: function(el, value){}
                  unbind: function (el){
                      el.removeEventListener('input', this.publish)
                  }
				}

Formatters

One-way formatters

				rivets.formatters.date = function(value){
					return moment(value).format('MMM DD, YYYY')
				}
<span rv-text="event.startDate | date"></span>

Formatters

Two-way formatters

				rivets.formatters.currency = {
					read: function(value) {
						return (value / 100).toFixed(2)
					},
					publish: function(value) {
						return Math.round(parseFloat(value) * 100)
					}
				}
<span rv-text="event.startDate | date"></span>

Formatters

Formatter arguments

				rivets.formatters.time = function(val, tz, format){
					return moment(value).tz(tz).format(format)
				}
<span>{data.time | time user.tz 'hh:mm'}</span>
