npm - пакетный менеджер для JavaScript

Что такое пакетный менеджер?

*Показывает на примере jQuery*

Пакет/package/module/component - готовое решение нашей задачи.

Пакетный менеджер помогает не только получать готовые решения,
но и обмениваться ими.

Модуль должен решать одну задачу и решать эту задачу хорошо.

Bower

Установка

npm install -g bower

"Что такое bower?"

- Что такое bower?
- Это пакетный менеджер. Устанавливается с помощью npm.
- Что такое npm?
- Это пакетный менеджер.

Главное отличие от npm

bower - плоская структура зависимостей

npm - древовидная

Главное отличие от npm

A > C

B > C

npm install

					node_modules
					|_A
					| |_C
					|_B
					  |_C

bower install

					bower_components
					|_A
					|_B
					|_C

Поэтому

bower - для клиента

npm - для сервера

но не всегда

bower install

bower install jquery 
			our-project
			|_bower_components
				|_jquery
					|_src/
					|_package.json
					|_...

Cobalt

Cobalt

Составляющие

  • Скаффолдинг (Slush)
