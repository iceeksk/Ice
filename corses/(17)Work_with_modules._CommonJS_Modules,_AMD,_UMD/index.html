Модулі

Дехтяренко Михайло

Модулі в JS

Що?

				<script src="lib/jquery.js"></script>
				<script src="lib/knockout.js"></script>
				<script src="lib/datepicker.js"></script>
				<script src="custom/handlers.js"></script>
				<script src="setup.js"></script>

Чому?

  • Немає єдиної точки входу
  • Збільшення складності
  • Неявні зв'язки між модулями

Визначення

Модуль — функціонально закінчена частина коду, який виконує певну задачу.

Модульне програмування — організація системи як сукупності невеликих незалежних блоків.

Навіщо це потрібно?

  • Спрощення системи
    • Рефакторинг
    • Семантика
    • Структурованість
  • Повторне використання модулів

Правила

  • Ніяких глобальних змінних
  • Модуль декларує API
  • Нічого не робити, поки не попросять

Рішення?

  • Імітація
  • CommonJS Modules
  • AMD
  • ECMAScript 6 Modules

Приклади?

  • CJS Modules: основа NodeJS, browserify для браузерів
  • AMD: require.js для браузерів
  • ES6 Modules: майбутнє JS

Імітація

				myCalc = (function(){
				    return {
				        calcSomething: function(number){
				            return math.calcThis(number);
				        }
				    };
				}());

Імітація

				(function(){
				    var myCals = {};
				 
				    myCals.calcSomething = function(number){
				        return math.calcThis(number);
				    };
				 
				    window.myCalc = myCalc;
				}());

CommonJS Modules

  • module, exports, require
  • Потрібне середовище збірки (browserify)
  • Не потрібна обгортка

CommonJS Modules

				var math = require('math');
				 
				exports.calcSomething = function(number){
				    return math.calcThis(number);
				};

Async Module Definition

  • define
  • Потрібно скачати та підключити бібліотеку(require.js)
  • Сам по собі надає обгортку для модуля
  • Асинхронне завантаження

Async Module Definition

				define(['math'], function(math){
				    return {
				        calcSomething: function(number){
				            return math.calcThis(number);
				        }
				    };
				});

ECMAScript 6 Modules

  • module, import, export (ключові слова)
  • Вже підтримуєтсья деякими браузерами

ECMAScript 6 Modules

				import 'math' as math;
				 
				export function calcSomething(number){
				    return math.calcThis(number);
				}

Теорія

  • Low Coupling & High Cohesion
  • Layered Architecture
  • Law of Demeter

Low Coupling & High Cohesion

  • Мала кількість залежностей
  • Слабка залежність від змін
  • Висока міра повторного використання

Layered Architecture & Law of Demeter

  • Модуль володіє обмеженим знаннями про інші модулі
  • Модуль звертаєтсья тільки до модулів свого рівня чи рівня нижче
