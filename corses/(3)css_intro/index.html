W3C и CSS

  1. CSS 1 (1996, 1999) - параметри шрифтів, кольори, ...
  2. CSS 2 (12 травня 1998), CSS 2.1 (07 червня 2011) - блочна верстка, селектори, ...
  3. CSS 3 знаходиться в стадії доопрацювання - трансформації, анімації, ...
  4. CSS 4 розробляється з 29 вересня 2011 року.

Що ж таке CSS?

Каскадні таблиці стилів.

body{
	font-family: Arial;
	background: #000;
}
#header{
	text-align: left;
	width: 210px;
}

Переваги CSS

  1. Відділяє контент (структуру) від оформлення(представлення)
  2. Розширює способи оформлення элементів
  3. Дає єдине стильове оформление безлічі документів
  4. Прискорює завантаження сайта
  5. Дозволяє створювати різне оформлення для різних пристроїв

Підключення стилей

Підключити стилі на сторінку можна додавши код в тег <head> HTML сторінки


Існує 3 способа підключення стилів на сторінку

Зовнішні (зв'язані) таблиці стилей

Використовуються для стилізації великої кількості сторінок

<link rel="stylesheet" media="all" href="css/styles.css"/>

Внутрішні (глобальні) таблиці стилів (фу-фу-фу)

Використовуються для стилізації однієї сторінки, до використання не рекомендуєтся

<style>
h1{
	color:red;
}
</style>

Вбудовані стилі (безпосередній опис стилів в элементі)

Використовуються рідко, в тих випадках коли по іншому ніяк (динаміка яку невзмозі забезпечити додавання класу)

<p style="color: red">Some text</p>

Імпорти

Можливість імпортувати безліч css файлів використовуючи @import


@import: url('reset.css');
@import: 'reset.css' screen, print;

Структура CSS правила

CSS правило складається з селектора та опису

selector [, selectors]{
	property: value [!important];
}

Види CSS селекторів

  1. Селектори тегів
    2. body{
	margin: 0 auto;
}
  2. Селектори по ідентифікатору (#)
    3. #page{
	background: #e1e1d5;
}

Види CSS селекторів

  1. Селектори по класу (.)
    2. .entry-title{
	color: #999966;
}
  2. Глобальні селектори (*)
    3. *{
	color: red;
}

Види CSS селекторів

  1. Селектори вкладеності
    2. div p{
	text-align: center;
}
  2. Селектори по атрибуту ([])
    3. a[href], a[target=_blank], [title~=flower], [lang|=en],
a[href^="https"], a[href$=".pdf"], a[href*="w3schools"]{
	color: green}

Види CSS селекторів

  1. Селектори по прямим потомкам
    2. div > p{
	color: red;
}
  2. Селектори по рівню (siblings)
    3. div ~ p{
	text-align: center;
}

Види CSS селекторів

  1. Слідуючий по рівню 
    2. div + p{
	color: green;
}
  2. Селектори псевдокласів по DOM станам
    3. li:first-child, li:nth-child(a*n+b),
    li:nth-of-type,a:visited{
	color: blue; }

Види CSS селекторів

  1. Селектори псевдоелементів (css3)
    2. p::first-letter, p::first-line, li::before{
	padding: 20px;
}

Псевдокласи посилань

  • :link – невідвідані посилання​
  • :visited – відвідані посилання

Працюють тільки з тими посиланнями які мають 'href' атрибут.

:link - може бути надмірним.

Псевдокласи дій користувача

  • :hover – застосовується коли користувач наводить курсор на елемент
  • :active – застосовується коли користувач утримує кнопку миші на елементі
  • :focus – застосовується коли на елементі встановлено фокус (по кліку чи за допомогою клавіши 'tab')​

По замовчуванню фокус можуть приймати інтерактивні елементи (a, button, input, textarea, select … )​.

Атрибут tabindex дає можливість приймати фокус будь-яким елементам.​

Сила каскадності!!!

Використання на одній сторінці різних таблиць стилів. Інтерпретація стилів по порядку, по деяким правилам каскадності (зв'язані, внутрішні, вбудовані) з врахуванням специфічності

Економія часу з наслідуваням

Наслідування означає, що якщо не вказано інше правило, то конкретний стиль будет унаслідований іншими елементами сторінки HTML.

<body style="font-family: Arial">
	<p>This text will be Arial as well</p>
</body>

*Замість стилізації кожного параграфа окремо, можна визначити колір шрифта для тега body
