Плавные переходы и анимации

Плавные переходы. Определение

CSS-переходы позволяют постепенно (плавно) изменять значения свойств на определенном временном отрезке.

Как это работает

					 
					<div></div>
					 
				
					div{
					 width: 100px;
					}
					div:hover{
					 width: 200px;
					}

Как это работает с переходами

					 
					<div></div>
					 
				
					div{
					 width: 100px;
					transition: 0.5s;
					}
					div:hover{
					 width: 200px;
					}

Свойство transition

			transition-property: all;// width, opacity, color...
			transition-duration: 2.3s;// 0, 134ms, 0.3s...
			transition-timing-function: ease;// linear, ease-in...
			transition-delay: 500ms;// 3s, 230ms...
			 
			transition: width 2.3s linear 500ms;

Анимации. Определение

Манипулируйте значениями свойств, используя кейфреймы. Описывайте поведение анимаций, управляя их продолжительностью, кол-вом повторений и конечным состоянием.

Как это работает

					 
					<div></div>
					 
				
					div{
					 width: 100px;
					}
					div:hover{
					 width: 200px;
					}

Как это работает с анимацией

					 
					<div></div>
					 
				
					div{
					 ...
					}
					div:hover{
					animation: boo 1s;
					}

Кейфреймы

			@keyframes boo{
			 from{
			  transform: translateX(0px) rotate(0deg);
			 }
			 50%{
			  transform: translateX(100px) rotate(90deg);
			 }
			 to{
			  transform: translateX(0px) rotate(180deg);
			 }
			}

Свойство animation

			animation-name: boo;
			animation-duration: 0.5s;
			animation-timing-function: linear;
			animation-iteration-count: 5;
			animation-direction: normal | reverse | alternate;
			animation-play-state: running | paused;
			animation-delay: 0.3s;
			animation-fill-mode: forwards | backwards | both;
			 
			animation: boo 2s linear 0.5s infinite;

Трансформации

Определение

Трансформации позволяют преобразовывать элементы в двухмерном и трехмерном пространствах.

Свойство transform

			element{
			 width: 30px;
			 height: 30px;
			 ...
			transform: translateY(10px) rotate(20deg);
			}

2D

Двухмерные трансформации

Точка, относительно которой...

			transform-origin: 50% 50%; // дефолт
			transform-origin: 123px 80%;

3D

Трехмерные трансформации

Дополнительные свойства

  • perspective
  • transform-style
  • backface-visibility

Перспектива

			perspective: 1000;
			perspective: 125px;

Как использовать

			.wrapper
			 .block
		
			.wrapper{ perspective: 1000px; }
			.block{
			 transform: rotate3d(0, 1, 0, 40deg);
			}

3D-контекст

			transform-style: preserve-3d;
			transform-style: flat;

Как использовать

			.outer-wrapper
			 .inner-wrapper
			 .block
		
			.outer-wrapper{ perspective: 1000px; }
			.inner-wrapper{ transform-style: flat; }
			.block{ transform: rotate3d(0, 1, 0, 40deg); }

Обратная сторона элемента

			backface-visibility: visible;
			backface-visibility: hidden;
